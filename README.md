# Wang-Bot
Discord bot for fun commands like /hunt, /equip, /daily

## Commands
- /ping - Check bot latency
- /help - Show all commands
- /info - Server info
- /hunt - Start hunting
- /equip - Equip item
- /daily - Collect daily reward

## Setup
1. Clone repo
2. Add `.env` with `DISCORD_TOKEN`
3. Run `node index.js`
